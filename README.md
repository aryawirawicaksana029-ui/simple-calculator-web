# Simple Calculator

A simple calculator web app built while learning the fundamentals of HTML, CSS, and JavaScript. This is a learning project — built to understand how the three core web languages work together, coming from a Python background.

## Features

- Basic operations: addition, subtraction, multiplication, division
- Clear (C) and delete last digit (DEL) buttons
- Divide-by-zero protection (shows an error message instead of crashing)
- Dark mode toggle
- Responsive layout

## What I Learned

- **HTML** structures the page — buttons, display, layout containers (`<div>`)
- **CSS** controls the look — colors, spacing, grid layout for the button pad, dark mode styling via class toggling
- **JavaScript** handles the logic — tracking input state with variables, doing calculations, connecting button clicks to functions via `onclick` and `addEventListener`

Coming from Python, the logic side (variables, functions, if/else) felt very familiar — the syntax is just different. The new concept was understanding how HTML, CSS, and JS are separate files/languages that talk to each other through `class`, `id`, and event listeners.

## Tech Stack

- HTML5
- CSS3 (Flexbox & Grid)
- Vanilla JavaScript (no frameworks/libraries)

## How to Run

Just open `calculator.html` in any browser — no installation needed.
